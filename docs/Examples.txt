Paste code into file "Runner.kt", then see the transpiled output in the stdout.
Function calls look like "arg1 .func arg2", and functions are defined as

    fn functionName param1 param2 {
        return param1 + param2
    }

Example program below:

    aw = "asdf "
    aw + "bcjk"
}

scopeAssignment .print
